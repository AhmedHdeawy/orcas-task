# Assessment Solution

## Features

- Using Abstarct Factory Pattern.
- Using Repository Pattern.
- Unit tests coverage.
- Handle all Acceptance Criteria.

## Installation

```sh
composer install
php artisan k:g
php artisan config:cache
php artisan serve
```
